J-Star

sq_norequest(opts)

この関数はやや特殊です。 挙動はsq_post系とよく似ていますが、実際にはサーバにアクセスしません。
この関数で実行されることは、まず1．ボタンが押下された際にフォームの入力チェックを行います。 そして、2．フォームのデータをJavascriptのオブジェクトに変換し、3．ユーザが定義した関数を呼び出します。 sq_post系の関数では2と3の間でサーバにアクセスしますが、この関数ではアクセスされません。

この関数で設定できるオプションは以下の通りです。（sq_post系と違いURLの指定は意味がありません）

callback
必須。ボタンを押下した際に実行される関数です。この関数は第1引数にフォームデータが渡されます。
form_id
任意。指定されなかった場合、押下するボタンが存在するフォームのIDを自動で取得します。
form2obj
任意。フォームに対して入力チェックを実行し、フォームデータをJavascriptに変換する関数です。 デフォルトはjQuery.sq.form2objです。
