valid_message(val,args)

validateでエラーが発生した際、表示するメッセージを設定します。

valid_message(val)との違いは、固定メッセージではなく動的なメッセージを設定可能なことです。
このvalid_message関数の第1引数は、メッセージオブジェクトのキーになります （メッセージオブジェクトに関しては、XXXをご覧下さい）。 そして第2引数はそのメッセージに設定する値になります。 例えば、メッセージが以下のような場合 

jQuery.sq.message({
  sample_message: "{0} は {1}です."
});
次のように使用します。 
  valid_message("sample_message" ["foo","bar"]);
こうすると表示されるメッセージは「foor は barです.」となります。

その他、表示されるタイミングなどはvalid_message(val)と同一です。
