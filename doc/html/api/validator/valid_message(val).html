valid_message(val)

validateでエラーが発生した際、表示するメッセージを設定します。

テキストフィールドのchangeイベントで入力チェックが実施された場合はTIPとして表示されます。 デフォルトはそのフィールドの右側に出力されます。 これは、jquery.suziQ.settings.js で設定できます。 

jQuery.sq.settings.validate_tip=true;
jQuery.sq.settings.validate_tip_position='right';
validate_tipをfalseにするとchangeイベント時にメッセージは出力されません。

またformデータを送信する直前の入力チェック時には、まとめて画面に出力されます。 出力される場所は、idがerrというノードに出力されます（これもjquery.suziQ.settings.js で設定可能です）。 また、id=errのノードが見つからない場合、チェックしたformの直前に出力されるようになります。
