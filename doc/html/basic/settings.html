J-Star

settings.js

J-Starでは「jquery.suziQ.settings.js」で設定を変更することができます。 利用可能な設定は以下の通りです。

国際化に関する設定

jQuery.sq.settings.locale_path
ロケールを決定するサービスのURLを設定します。
この値がnull（あるいはundefined）になっている場合、国際化機能がOFFになります。
jQuery.sq.settings.message_charset
メッセージファイルを読み込むときのcharsetを設定します。
jQuery.sq.settings.locale_currency
ロケールに合わせた通貨記号を設定します。ロケールをキーに、通貨記号を値にしたオブジェクトを設定してください。

入力チェックに関する設定

jQuery.sq.settings.validate_err_class
入力チェックエラーになったときに、使用されるCSS
jQuery.sq.settings.form_validate_message_area_selector
フォーム全体の入力チェックでエラーがでた場合に出力されるノードのCSSセレクター
デフォルトは「#err」。このセレクターでノードが見つからない場合、フォームの先頭にエラーメッセージが表示されます。
jQuery.sq.settings.validate_tip
テキストフィールドに入力した際に実行される入力チェックでエラーが起きた際に、バルーンを出すかどうかを決定します。 trueの場合、出力されます。
jQuery.sq.settings.validate_tip_position
入力エラーのバルーンがどこに出るのかを設定します。デフォルトは「right」です。

ajaxに関する設定

jQuery.sq.settings.ajax_dataType
jQueryのajax関数のdataTypeの値を設定します。デフォルトは「text」です。


その他の設定

jQuery.sq.settings.debugger_keycodes
デバッガーを起動するためのキーコードを設定します。デフォルトは [16,17,18,32]　という配列です。 これはWindowsでは、「Ctrl + Shift + Alt + Space」です。
