J-Star

J-Star,SuziQの構成

J-StarはOSSのjQuery,SuziQ をベースとしたFWです。
SuziQはAjaxをモック化するためのツールで、J-StarはこのSuziQをより便利に使うためのFWになっています。 J-Starでは以下をプラグインとして提供しています。

またこのドキュメントでは、J-StarとSuziQは結びつきが大変強いためあまり2つを区別せずに記述します。


[図1]全体の構成図


Formatter
入力されたタイミングで入力値のフォーマットを行います。文字列から日付、数値、金額への変換が容易に実現できます。

Validation
入力値のバリデーションを行います。必須、文字種、文字長、値の存在範囲、などがチェックできます。正規表現、独自実装のバリデーションも可能です。


Input prompt
入力欄に初期表示で入力を促すメッセージを表示できます。cssクラスを独自に定義することも可能です。


DOM操作
dom操作の簡易なAPIを提供します。


Debugger
htmlファイルの状態や、変数に入力された値を確認することが出来ます。またフォーム･リンク項目の詳細も一覧表示できるdocumentationもあります。
