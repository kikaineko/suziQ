J-Star

to(url,data,selector(option))

擬似画面遷移を引き起こす関数です。

url
必須。遷移先画面のURLを相対パスで指定してください。
data
必須。次画面に渡すデータになります。これは次の画面のinit関数の第1引数に渡されます。
selector
任意。現在の画面のどのノードに対して書き換えを行うかを指定します。 デフォルトは「body」です。bodyノードを書き換えるため画面遷移を実行しているように見えることになります。


戻り値
jQuery
jQueryオブジェクト
ターゲットオブジェクト
SuziQオブジェクト
サンプル
サンプル1
$$.ajax.to("../hoge.html",{a:"123"});

このようにすると、../hoge.htmlに画面遷移をしhoge.htmlのinit関数に「{a:"123"}」のデータを渡します。
