fill(data,selector_fnc(option))

data中の値を、指定されたノードに書き込みます。
例えば

data={a:"1",b:"2"}
$('body').fill(data);

と書けば、bodyタグ以下の子ノード（bodyノードも含む） で、name属性がaというノードとbというノードに「1」と「2」の値を書き込みます。 仮にname=aのノードがinputであった場合、valueに「1」という値を設定します。 また、name=bのノードがdivなどのタグであった場合、innerHTMLに値を書き込みます。

data
必須。設定したいデータオブジェクト。
selector_fnc
任意。値を書き込むノードを指定する関数。
デフォルトはname属性とオブジェクトのキーが一致するノードを探索します。
