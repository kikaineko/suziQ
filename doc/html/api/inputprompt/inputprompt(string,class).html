J-Star

inputprompt(val, class)

入力欄内にあらかじめ文字を表示させたい場合に使用します。カーソルを合わせると文字は消えます。 デフォルトのCSSクラス（input_prompt）を利用せず、指定したCSSクラスを利用します。

戻り値
jQuery
jQueryオブジェクト
ターゲットオブジェクト
jQueryオブジェクト（inputpromptを設定したいノード）
サンプル
サンプル1
名前   <input type="text" name="sample" size=30/>

.sample_css { color: #0000FF;}

$("[name=sample]").input_prompt("ここに名前を入れてください。","sample_css");
