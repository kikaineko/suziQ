J-Star

createOptions(arr)

配列に格納された値をセレクトボックスタグで使用できるoption要素に変換します。

引数
arr
配列
valueとlabelを要素に持つオブジェクトの配列
戻り値
配列
option要素の配列
ターゲットオブジェクト
SuziQオブジェクト
サンプル
サンプル1
  data=[{value:"A00",label:"AAAA"},{value:"B01",label:"BBBB"}];
  $$.createOptions(data);

とすると、以下の配列オブジェクトが取得できます
[<option value="A00">AAAA</option>, <option value="B01">BBBB</option>]
