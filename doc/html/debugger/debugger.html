J-Star

debugger

SuziQは強力なデバッガーを備えています。

使い方

デバッガーを利用するには、本番モード以外のモードで
SHIFT + CTRL + ALT + SPACE
を同時に押してください。現在の画面にデバッガー画面が表示されます。
起動キーを変えたい場合は、jquery.suziQ.settings.js 内の jQuery.sq.settings.debugger_keycodes を上書きしてください。

機能説明


LOG
jQuery.sq.log関数に渡した文字列が出力されます。
これは、今までJavascriptのデバッグで用いられていたalert関数の代替となる機能です。
またメッセージの最後にタイムスタンプが出力されます。
このタイムスタンプは、new Date()オブジェクトの返すミリ秒です。

Javascript Runner
任意のJavascriptを実行できます。また、現在の画面情報が取得できます。サンプル画面で alert($("[name=name]").val()); と入力し実行してみてください。名前欄に入力した値が取得できます。このように、変数に代入された値を見ることができます。

Config
画面の基本情報が出力されます。
  • Basic Information
    • 現在のモード、ロケール、SuziQのバージョンが表示されます。

  • jQuery.sq.setup
    • setup関数内の下記情報を表示します。
    • messages
    • dummy_data
    • init関数

HTML
現在のHTMLが出力されます。まれにscriptタグの中身が出力されないことがあります。このため、なるべく重要な処理はinit関数の中で定義、実行してください。 また、関数名が分かっている場合は「Javascript Runner」で実装の中身を見ることも可能です。

documentation
画面情報が出力されます。デバッガー画面右上のdocumentationボタンを押すと表示されます。
  • Form Information
    • 画面内のフォーム情報です。
    Hilightをチェックすると該当項目をハイライト出来ます。

  • Bottun Information
    • 画面内のボタン情報です。
